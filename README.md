# Better Clouds is here
Better Clouds adds a new cloud block to the Minecraft 1.16.1 with Forge. The block spawns naturally in the cloud layer, and uses cellular automata rules to mimic real cloud changes.

## Getting started

1. Clone the repo: `git clone https://github.com/dannowilby/betterclouds.git`
2. Build the project: `./gradlew build`
3. Add to your mods folder!
